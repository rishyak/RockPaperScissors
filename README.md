# Rock Paper Scissors
To run this "brilliant" rock paper scissors engine, follow this guide.

## Usage
### Make venv
```
$ python3 -m venv .venv
```

### Activate venv
#### Linux/Darwin
```
$ source .venv/bin/activate
```
#### Windows
```
$ .venv\Scripts\activate
```

### Install Module
```
$ python3 -m pip install -e .
```

### Run
```
$ RockPaperScissors
```

### Deactivate venv
```
$ deactivate
```
